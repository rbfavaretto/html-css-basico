SELETORES:
    * h1, p
      A vírgula permite selecionarmos múltiplos elementos para a aplicação de um mesmo estilo.

    * p a
      Seleciona o a que tiver um p como elemento pai (não precisa ser filho direto) 

ID:
    Atributo html que adiciona um identificador  único na tag. Esse identificador pode ser utilizado no CSS para selecionarmos o elemento: #nomeid

CLASS:
    Atributo HTML que adiciona um identificador reutilizável na tag. Esse identificar pode ser utilizado no CSS para selecionarmos o(s) elemento(s): .class
